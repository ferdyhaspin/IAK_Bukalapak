# IAK_Bukalapak

This is a case study than activity **IAK** **( [`Indonesia Android Kejar`][2] )** year 2017.

## Features
- Get data json with [okhttp3][1]
- Detail product [bukalapak][0]
- Cek connection with boardcast android
- Notification
